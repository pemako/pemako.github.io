# 2017-02-02-jekyll-document

`jekyll` 依赖`ruby` 你的环境必须已安装`ruby`,使用`gem` 管理各种包信息。

## Install

安装`Jekyll` 事先要有相应的环境配置

* [`Ruby`](https://www.ruby-lang.org/)
* [`RubyGems`](https://rubygems.org/)
* `Linux, Unix, or Mac OSX`

如果环境都已经齐全运行下面的命令即可创建一个 site。

```text
gem install jekyll
jekyll new myblog # 创建一个 site
cd myblog
jekyll serve # 启动可以在浏览器中访问 http://localohost:4000查看效果
```
